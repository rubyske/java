## 1. Apa itu Pemrograman?

**Pemrograman** adalah proses menyusun instruksi logis menggunakan bahasa tertentu agar komputer dapat menjalankan tugas sesuai tujuan.

### Tujuan Pemrograman:

* Mengolah data
* Menyimpan & mengambil data
* Mengotomatisasi proses
* Menyediakan layanan (API)

> Intinya: **pemrograman adalah cara manusia berkomunikasi dengan komputer secara terstruktur**.

---

## 2. Apa itu Backend Developer?

**Backend Developer** bertanggung jawab atas sisi server dari sebuah aplikasi.

### Tanggung Jawab Utama:

* Mengelola logika bisnis
* Mengolah request & response
* Mengatur database
* Membuat API (REST / GraphQL)
* Autentikasi & otorisasi
* Menjaga performa & keamanan

### Contoh Fitur Backend:

* Login & registrasi
* Transfer saldo
* Hitung poin reward
* Proses pembayaran

> Backend adalah **otak dari sebuah sistem aplikasi**.

---

## 3. Perbedaan Backend dan Frontend

| Frontend                 | Backend                |
| ------------------------ | ---------------------- |
| Tampilan UI              | Logika bisnis          |
| Berjalan di browser      | Berjalan di server     |
| HTML, CSS, JavaScript    | Java, Python, Go       |
| Fokus ke user experience | Fokus ke data & proses |
| Tidak akses DB langsung  | Akses database         |

**Analogi:**

* Frontend = kasir
* Backend = sistem gudang & keuangan

## 5. Apa itu Java?

**Java** adalah bahasa pemrograman:

* Object-Oriented
* Berjalan di atas JVM (Java Virtual Machine)
* Prinsip utama: **Write Once, Run Anywhere**

### Digunakan untuk:

* Backend enterprise
* Banking & financial system
* Sistem transaksi besar
* Microservices

---

## 6. Mengapa Menggunakan Java?

Keunggulan Java:

1. Stabil & mature
2. Strong typing (minim bug runtime)
3. Ecosystem besar (Spring, Hibernate)
4. Performa konsisten
5. Mudah di-maintain oleh tim besar

> Java cocok untuk **sistem backend jangka panjang**.

---

## 7. Mengapa Java Populer?

Java populer karena:

* Digunakan di banyak perusahaan besar
* Banyak legacy system berbasis Java
* Dominan di sektor:

  * Banking
  * Fintech
  * Telco
  * E-commerce

> Banyak lowongan backend enterprise menggunakan Java.

---

## 8. Java di Era AI

Java **masih sangat relevan** di era AI.

### Peran Java dalam AI:

* Backend API untuk AI service
* Integrasi AI ke sistem existing
* High-performance microservices

**Catatan:**

* Python: training model AI
* Java: production & deployment

> Java unggul dalam **menjalankan AI di production system**.

---

## 9. Masa Depan Java

Java memiliki masa depan kuat karena:

1. Sistem enterprise tidak mudah diganti
2. Update rutin (Java 17, 21 LTS)
3. Spring Boot terus berkembang
4. Cocok untuk cloud & microservices
5. Demand backend Java tetap tinggi

> **Java bukan bahasa tren, tapi bahasa industri.**
